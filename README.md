## Описание

База для одностраничного лэндинга с возможностью телеграм оповещений


## Установка

```bash
# Установка пакетов PHP
$ composer install

# Установка пакетов JS
$ npm install

# Скомпилировать JS и CSS ресурсы для VITE
$ npm run build

# Копирование шаблона для env настроек проекта 
$ cp .env.example .env
```

## Запуск приложения

```bash
# В режиме разработки
$ php artisan serve
```
